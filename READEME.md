# LIFEMUSIC FRONTEND

## Contents

## Required Libraries

- Npm
  - Node.js >= 12.x
- Yarn 1.12

## How to run

```bash
  npm install -g webpack-dev-server   # install `webpack dev server` with global, only first time
  yarn install  # install libraries, only first time
  yarn build    # yarn build, only first time
  yarn start    # start dev server
```
